# Vue_ant_admin

#### &nbsp;&nbsp;本项目是 cli3 脚手架搭建而成。

#### &nbsp;&nbsp;集成了项目常用的工具包、axios 拦截器统一封装 api、上线打包优化方案

##### &nbsp;&nbsp;从公司项目剥离而出的初始化模板

<table><tr><td bgcolor=#C0C0C0> git clone https://github.com/17721524631/VueAntAdmin.git</td></tr></table>
